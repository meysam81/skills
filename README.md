# Meysam's Skills

Curated [Agent Skills](https://github.com/anthropics/agent-skills) for [Claude Code](https://claude.com/claude-code) by [@meysam81](https://github.com/meysam81).

## Install

Install a specific skill:

```bash
claude skills add meysam81/skills --skill google-golang-style
```

Install all skills:

```bash
claude skills add meysam81/skills
```
